# Python Game Stanford

Este proyecto es un juego desarrollado en Python, diseñado como parte de un curso de programación. El juego incluye lógica para manejar la entrada del usuario y la dinámica del juego.

## Estructura del Proyecto

El proyecto tiene la siguiente estructura de directorios:

``` cmd
PythonGameStanford
├── src
│   └── main.py          # Punto de entrada de la aplicación
├── tests
│   └── test_main.py     # Pruebas unitarias para main.py
├── requirements.txt      # Dependencias del proyecto
└── .gitignore            # Archivos y directorios a ignorar por Git
```

## Instalación

1. Clona el repositorio:

   ``` cmd
   git clone <URL_DEL_REPOSITORIO>
   ```

2. Navega al directorio del proyecto:

   ``` terminal
   cd PythonGameStanford
   ```

3. Instala las dependencias:

   ``` terminal
   pip install -r requirements.txt
   ```

## Uso

Para ejecutar el juego, utiliza el siguiente comando:

``` terminal
python src/main.py
```
